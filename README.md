#Brain Stroke Risk Prediction

A Machine Learning model to predict stroke risk using patient healthcare and lifestyle data. Implemented the K-Nearest Neighbors (KNN) algorithm to analyze factors such as age, BMI, glucose levels, hypertension, heart disease, and smoking status. Developed an interactive Gradio dashboard for real-time risk prediction and visualization.

Technologies Used
Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Imbalanced-learn (SMOTE), Matplotlib, Gradio, Joblib
Platform: Google Colab

Dataset Name: Healthcare Stroke Dataset

Features: Age, Gender, BMI, Average Glucose Level, Hypertension, Heart Disease, Smoking Status, Work Type, Residence Type, etc.

Workflow
Data Cleaning and Preprocessing
Label Encoding of Categorical Data
Data Balancing using SMOTE
Model Training using KNN
Stroke Risk Prediction
Dashboard Development using Gradio
Risk Visualization and Analysisv
